Endorphin.Instrument.Attocube.ANC300

Introduction

The Attocube ANC300 can be used to control a number of piezo stick-slip positioners. Two TCP ports are available for controlling the ANC300. A simple command socket allows control of voltage, frequency and applying a number of steps. Additional functionality is available via a Lua console.

This driver can communicate with both to control the piezo positioning and to perform short range piezo scanning.
